# FoodRecipesApp-MVVM

The application works with Spoonecular API "https://spoonacular.com/"

Technologies and Libraries Used :

* MVVM
* DI (Hilt)
* ViewBinding
* DataBinding
* Coroutines
* Retrofit
* Flow
* Livedata
* Caching 
* Room
* ViewPager2
* DiffUtil
* RecyclerView
* Motion Layout
* Contextual Action Mode
* Share Data with Other Apps
* DataStore Preferences
